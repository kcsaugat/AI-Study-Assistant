# 📚 USER MANUAL - AI Study Assistant

Welcome! This guide explains every feature of your AI Study Assistant.

---

## 🚀 Getting Started

### Create Your Account

1. Go to **https://ai-study-assistant.vercel.app**
2. Click **"Register"**
3. Enter:
   - **Email**: Your email address
   - **Password**: At least 6 characters (mix of letters, numbers, symbols recommended)
4. Click **"Sign Up"**
5. You're now logged in! 🎉

### Login

1. Go to **https://ai-study-assistant.vercel.app**
2. Click **"Login"**
3. Enter your email and password
4. Click **"Sign In"**

## 📝 Notes

### Create a New Note

1. Click **"+ New Note"** button
2. Enter a **Title** (e.g., "Biology Chapter 5")
3. Paste or type your **Study Content**
4. Click **"Save Note"**

### View Notes

1. Click **"Notes"** in sidebar
2. See list of all your notes
3. Click any note to open it

### Edit a Note

1. Open the note
2. Click **"Edit"** button
3. Change title or content
4. Click **"Save Changes"**

### Delete a Note

1. Open the note
2. Click **"Delete"** button
3. Confirm deletion

### Note Details Page

When you open a note, you see tabs:

- **📄 Content** - Your original text
- **✨ Summary** - AI-generated summary
- **❓ Quiz** - Questions to test yourself
- **🎯 Flashcards** - Study cards with answers
- **💬 Chat** - Ask AI about your note

## ✨ Summarize Notes

Instantly summarize long text into key points!

### How to Summarize

1. Open a note
2. Click **"Summarize"** button
3. Wait a few seconds
4. Read the summary in the **Summary tab**

### Features

- ✨ AI extracts key points
- 📊 Organized in bullet points
- 🔄 Regenerate for different summaries
- 📋 Copy summary to clipboard

### Tips

- Works best with well-structured text
- Try on textbook chapters, articles, research papers
- Regenerate if you want a different angle

---

## ❓ Generate Quiz

Create multiple-choice quizzes from your notes!

### How to Create a Quiz

1. Open a note
2. Click **"Generate Quiz"** button
3. AI creates 5-10 questions
4. Click **"Quiz"** tab to see questions

### How to Take the Quiz

1. Read the question
2. Click your answer choice
3. Click **"Next"** or **"Submit"**
4. See if you're correct
5. Get your final score

### Features

- 🎯 Multiple-choice format
- ✅ Instant feedback
- 📊 Score tracking
- 🔄 Regenerate for new questions

### Tips

- Take multiple times to improve
- Review incorrect answers
- Share your score with friends

---

## 🎯 Flashcards

Study with AI-generated flashcards!

### How to Create Flashcards

1. Open a note
2. Click **"Generate Flashcards"** button
3. AI creates a deck of cards
4. Click **"Flashcards"** tab to study

### How to Study

1. Read the **question** side
2. Click the card to **flip** and see answer
3. Click **"Easy"**, **"Medium"**, or **"Hard"**
4. Continue through the deck
5. Track your progress

### Features

- 🎴 Flip cards to reveal answers
- 📊 Progress tracking
- 🔄 Review difficult cards
- 📈 Spaced repetition algorithm

### Study Tips

- Study 15 minutes daily
- Focus on "Hard" cards
- Review before exams
- Create multiple decks for different topics

### Multiple Decks

- Each note creates a separate deck
- Review different subjects separately
- Keep decks organized by topic

## 🤖 AI Tutor Chat

Chat with an AI tutor about your study material!

### Start a Chat

1. Click **"Chat"** in sidebar
2. Click **"+ New Session"** to start conversation
3. Type your question
4. Click **"Send"** or press Enter
5. Get instant AI response

### Chat Features

- 💬 Real-time conversations
- 📚 Context about your notes
- 🔄 Session history
- 🎯 Follow-up questions
- 📝 Save conversations

### How to Use

Ask questions like:
- "Explain photosynthesis simply"
- "What are the main points from my biology note?"
- "Give me practice questions on this topic"
- "Why is this concept important?"

### Chat Tips

- Be specific in questions
- Ask follow-up questions
- Save helpful responses
- Use for clarification and practice

### Session Management

- **New Session**: Start fresh conversation
- **View History**: See past messages
- **Delete Session**: Remove conversation
- **Share Session**: Get link to conversation

## 🆘 Troubleshooting

### App Won't Load

1. Refresh the page (Ctrl + R)
2. Clear browser cache
3. Try different browser
4. Check internet connection

### Can't Login

1. Check email spelling
2. Verify caps lock is off
3. Reset password
4. Check browser cookies enabled

### Summarize Not Working

1. Make sure note has enough text (100+ words)
2. Check internet connection
3. Wait a few seconds for AI
4. Refresh and try again

### Quiz Won't Generate

1. Note might be too short
2. Try different note
3. Refresh page
4. Report bug on GitHub

### Chat Not Responding

1. Check internet connection
2. Try new session
3. Refresh page
4. Report on GitHub issues

## 📊 Study Tips

### Effective Note-Taking

1. Write in your own words
2. Use bullet points
3. Highlight important concepts
4. Add examples

### Summarization Strategy

1. Read original notes
2. Create AI summary
3. Review for missing points
4. Add your own notes

### Quiz Strategy

1. Take quiz immediately after note
2. Review incorrect answers
3. Take quiz again in 1 week
4. Track improvement

### Flashcard Strategy

1. Start with easy cards
2. Focus on medium cards
3. Master hard cards
4. Review all daily

### Study Schedule

- **Daily**: 15-30 minutes flashcards
- **2x Weekly**: Full quiz review
- **Weekly**: Summary review
- **Before Exam**: Intensive review
